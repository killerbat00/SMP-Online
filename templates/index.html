APIs

v1/api/outlook

Outlook
/v1/api/outlook?age=age

age: The age of the user in years.

Returns the following:

{
 "age": the given age,
 "financial_security": calculated value of the user's financial security
 "description": a text description of the financial outlook
}

v1/api/spending

Spending
/v1/api/spending?age=age&daily_spending=daily_spending

age: The age of the user in years.
daily_spending: The amount of money the user spends per day in dollars.

Returns the following:

{
 "age": the given age,
 "daily_spending": the given daily spending,
 "years_before_financial_ruin": the # of years before financial ruin,
 "remaining_late_night_amazon_sprees": the # of remaining 2am half-eyed amazon shopping sprees (avg of $100/spree),
 "est_financial_security_driven_panic_attacks": est # of remaining panic attacks driven by the calculated financial security,
}
